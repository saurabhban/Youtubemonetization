WHAT THIS APP DOES
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
1. You type a topic (or pick from AI-generated ideas)
2. GPT-4o writes a full 8-minute video script (Indian audience)
3. Microsoft Edge TTS generates a professional voiceover (FREE)
4. Pexels API downloads matching stock video clips (FREE)
5. FFmpeg assembles: title card + clips + captions + outro
6. Video uploads directly to your YouTube channel

Target niche: Personal Finance India (CPM ₹100–₹250 — HIGHEST in India)

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
STEP 1 — PREREQUISITES (install once)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
→ Python 3.10+     https://python.org/downloads
→ FFmpeg            https://ffmpeg.org/download.html
  (Mac: brew install ffmpeg)
  (Windows: download from ffmpeg.org, add to PATH)

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
STEP 2 — GET API KEYS (all free to start)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
A) OpenAI API Key (for GPT-4o script writing)
   → Sign up: https://platform.openai.com/
   → Go to: API keys → Create new key
   → Cost: ~₹3-5 per video script (very cheap!)

B) Pexels API Key (for FREE stock footage)
   → Sign up: https://www.pexels.com/api/
   → Completely FREE (200 requests/hour)

C) YouTube API (to upload videos to your channel)
   → Go to: https://console.cloud.google.com/
   → Create a new project (e.g. "MyYouTubeBot")
   → Library → Search "YouTube Data API v3" → Enable
   → Credentials → Create → OAuth 2.0 Client ID → Desktop App
   → Download the client_secrets.json or note Client ID + Secret

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
STEP 3 — INSTALL & CONFIGURE
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
1. Open Terminal in this folder
2. Run:  bash setup.sh
3. Edit the .env file with your API keys:
   OPENAI_API_KEY=sk-...
   PEXELS_API_KEY=...
   YOUTUBE_CLIENT_ID=...
   YOUTUBE_CLIENT_SECRET=...
   CHANNEL_NAME=YourChannelName

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
STEP 4 — RUN THE APP
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
  python webapp.py
  → Opens dashboard at http://localhost:5000
  → Click "Connect YouTube" (one-time browser login)
  → Pick a topic → Click "Generate & Upload Video"
  → Watch the progress bar — takes 5-15 minutes per video!

STEP 5 — CLI USAGE (advanced)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
